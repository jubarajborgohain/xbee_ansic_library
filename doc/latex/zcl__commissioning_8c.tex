\subsection{Detailed Description}
Implementation of Z\-C\-L Commissioning Cluster. For devices with limited resources (e.\-g., P\-X\-Bee), we could have a \char`\"{}minimal\char`\"{} \hyperlink{structzcl__comm__startup__param__t}{zcl\-\_\-comm\-\_\-startup\-\_\-param\-\_\-t} structure and zcl\-\_\-comm\-\_\-startup\-\_\-attributes object, similar to how we have \char`\"{}base\char`\"{} and \char`\"{}full\char`\"{} attribute types.

It would be necessary to have a flag though, so either the minimal or full structure can be passed to functions that process it.
