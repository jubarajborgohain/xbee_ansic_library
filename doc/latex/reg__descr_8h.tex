\subsection{Detailed Description}
Header for X\-Bee register descriptors. \begin{DoxyNote}{Note}
This header is sensitive to the value of the macro I\-D\-I\-G\-I\-\_\-\-U\-S\-E\-\_\-\-X\-B\-E\-E. If this macro is defined, some additional fields in the register descriptor structure are defined (and initialized). This data is required for i\-Digi support to be included. 
\end{DoxyNote}
