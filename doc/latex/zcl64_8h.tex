\subsection{Detailed Description}
Macros for working with 64-\/bit integers via the zcl64\-\_\-t datatype. Makes use of the \hyperlink{struct_j_s_int64}{J\-S\-Int64} type and support functions included in the platform support code (\hyperlink{jslong_8h_source}{jslong.\-h}, \hyperlink{jslong_8c}{jslong.\-c}).

On some platforms (Win32), zcl64\-\_\-t is simply a uint64\-\_\-t. Most embedded platforms represent a zcl64\-\_\-t value as a structure. Because of that difference, you can't just write \char`\"{}c = a + b\char`\"{} or \char`\"{}a.\-lo = $\sim$a.\-lo, a.\-hi = $\sim$a.\-hi\char`\"{} -- neither statement is portable to the other platform type.

Therefore, it is necessary to use the macro functions in this file to manipulate zcl64\-\_\-t variables.
