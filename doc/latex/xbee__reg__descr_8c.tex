\subsection{Detailed Description}
Code related to the register descriptor table. The X\-Bee register descriptor table is initialized here.

\begin{DoxyNote}{Note}
\hyperlink{sxa_8h}{sxa.\-h} needs to be included before including this header.

This header is sensitive to the macro I\-D\-I\-G\-I\-\_\-\-U\-S\-E\-\_\-\-X\-B\-E\-E. If defined, then extra fields are included in the table in order to support i\-Digi. Otherwise, the fields are omitted so as to save storage. 
\end{DoxyNote}
