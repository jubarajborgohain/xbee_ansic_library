\subsection{Detailed Description}
Code related to I/\-O sampling and configuration (the A\-T\-I\-S command, 0x92 frames). \begin{DoxyNote}{Note}
This code is not directly concerned with collecting I/\-O samples. To do that, it is necessary to register a command handler for A\-T\-I\-S commands, and/or appropriate endpoints to handle the incoming data frames. If not using the Simple X\-Bee A\-P\-I (S\-X\-A) layer, then see \hyperlink{xbee__sxa_8c}{xbee\-\_\-sxa.\-c} for examples of how to register the necessary callback functions. Otherwise, it is recommended to use S\-X\-A for all I/\-O manipulation. 
\end{DoxyNote}
